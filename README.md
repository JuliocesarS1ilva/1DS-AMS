# 1DS-AMS

Este repositório reúne atividades desenvolvidas durante o processo de aprendizagem em programação, com foco em **Java**, **desenvolvimento web (HTML/CSS)** e **JavaScript**.

---

## Sobre

Este repositório tem como objetivo organizar exercícios e pequenos projetos desenvolvidos para prática e fixação de conceitos fundamentais de programação.

Ele serve como base de estudo e evolução contínua em diferentes tecnologias.

---

## Conteúdo do repositório

- **Java** → Exercícios de lógica de programação e fundamentos da linguagem  
- **HTML/CSS** → Estruturação e estilização de páginas web  
- **JavaScript** → Lógica aplicada e criação de uma calculadora funcional  

---

## Tecnologias utilizadas

- Java  
- HTML5  
- CSS3  
- JavaScript
